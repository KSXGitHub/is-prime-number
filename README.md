# IsPrimeNumber JavaScript function
---

## Installation
`npm install is-prime-number`

## Usage
Node.JS:
```javascript
const {isPrime} = require('is-prime-number');
console.log( isPrime(7) ); // true
console.log( isPrime(20) ); // false
```

ES6:
```javascript
import {isPrime} from 'is-prime-number';
console.log( isPrime(7) ); // true
console.log( isPrime(20) ); // false
```
